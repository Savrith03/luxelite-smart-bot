# Luxelite Group - Smart Guest Service AI Bot

This repository contains a prototype implementation for an **AI-powered Guest Service Telegram Bot** designed specifically for Luxelite Group's hospitality properties, including Palace Gate Hotel & Resort, Eden Beach Resort, and Royal Dim Sum.

## Features
- **Intent Classification:** Employs rule-based Natural Language Processing (NLP) to detect guest core requirements (Booking, Check-in times, Amenities, and Dining).
- **Multi-Property Logic:** Easily scalable to route configurations across all 19+ Luxelite subsidiaries.
- **Workflow Automation:** Integrates customer support touchpoints, reducing human operational dependency and standardizing information flow.

## Tech Stack
- Python 3
- Native Telegram Bot API (python-telegram-bot)
